================================================================================
COMMON ISSUES
================================================================================

ISSUE: Page looks the same (v6 styling still showing)

FIX:
  1. Hard refresh: Cmd+Shift+R
  2. Empty cache: DevTools → Application → Storage → Clear all
  3. Verify base.html replaced: Check file size (28KB, not 24KB)
  4. Restart Flask: Ctrl+C then python run.py


ISSUE: Text is still grey instead of Aura Indigo

FIX:
  1. Check base.html contains: color: #B0A6DF
  2. Hard refresh browser
  3. If still grey, check DevTools → Styles tab
  4. Verify color property value is #B0A6DF


ISSUE: Background looks different

FIX:
  1. Verify background color: body should show background-color: #0f0520
  2. Check for background-image with multiple radial-gradients
  3. Ensure starfield is loading (small scattered dots)


ISSUE: Buttons don't glow on hover

FIX:
  1. Check System Preferences → Accessibility → Display
  2. Turn OFF "Reduce motion" if enabled
  3. Hard refresh browser (Cmd+Shift+R)
  4. Hover over button and wait for animation
