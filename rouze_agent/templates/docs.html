Rouze Intelligence API Documentation

Authentication

All API requests require an API key:

X-API-Key: your_api_key_here

Demo key: demo_key_123

POST /api/v1/analyze

Analyze raw signals for intelligence

Request:

curl -X POST http://localhost:8000/api/v1/analyze \
  -H "Content-Type: application/json" \
  -H "X-API-Key: demo_key_123" \
  -d '{
    "signals": [
      {"text": "Patient experiencing severe headaches", "source": "forum"}
    ]
  }'

Response:

{
  "status": "success",
  "analysis": {
    "sentiment": "negative",
    "risk_score": 87
  }
}

GET /api/v1/usage

Check API usage

Request:

curl http://localhost:8000/api/v1/usage \
  -H "X-API-Key: demo_key_123"

Response:

{
  "tier": "professional",
  "calls_used": 42,
  "calls_remaining": 4958
}
